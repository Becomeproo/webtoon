## ℹ️앱 설명

웹 뷰를 활용하여 네이버 웹툰을 보고 최근 웹툰을 저장할 수 있는 앱

<br>

## ✅구현 기능

* ViewPager와 Fragment를 사용해 3개의 페이지 구현
* SharedPreference를 활용해 최근 페이지 등록
* Dialog를 활용해 Tab 이름 설정

<br>

## ✅사용되는 기능

* Android
    * ViewPager2
    * TabLayoutMediator
    * WebView
    * Fragment
    * SharedPreference
    * Dialog
